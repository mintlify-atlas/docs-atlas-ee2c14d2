Answers to the most common questions about setup, compatibility, and behavior of the Edpuzzle Answers bookmarklet.

<AccordionGroup>
  <Accordion title="Why doesn't the popup open?">
    Make sure popups are allowed from `edpuzzle.com` in your browser settings. Also confirm that the CSP bypass extension is installed and active for the `edpuzzle.com` domain.

    If you are on Canvas or Schoology, you need to click the bookmarklet twice: once while on the Canvas or Schoology page (which opens the Edpuzzle assignment in a new tab), and again on the Edpuzzle tab to launch the full script.
  </Accordion>

  <Accordion title="I see 'Could not connect to any of the mirrors'. What does that mean?">
    The script attempts to load itself from two mirror URLs: `edpuzzle.hs.vc` and `edpuzzle.librecheats.net`. If both are unreachable, it means those domains are likely blocked on your network — school and institutional Wi-Fi often filters these.

    Try switching to a different network or using a VPN to bypass the network-level block.
  </Accordion>

  <Accordion title="Does this work on private assignments?">
    No. Private assignments return a 403 error from the Edpuzzle API, and the script cannot access answer data for them. This is an intentional restriction on Edpuzzle's side, and there is no way for the bookmarklet to work around it.
  </Accordion>

  <Accordion title="Will my teacher know I used this?">
    The script submits answers through the standard Edpuzzle API, so the submission looks identical to one made manually. There is no visible indicator in Edpuzzle's interface that the answers were generated by a tool.

    That said, completing an assignment very quickly may draw attention. The video skip and auto-answer features produce the same API footprint as a normal submission, but unusually fast completion times can raise suspicion depending on how closely the assignment is monitored.
  </Accordion>

  <Accordion title="Which browsers are supported?">
    Any modern browser that supports bookmarklets: Chrome, Edge, Firefox, Opera, and Brave. The CSP bypass extension required for the script to function is available for Chromium-based browsers and Firefox.
  </Accordion>

  <Accordion title="Why do I need a teacher account to run the server?">
    The media proxy endpoint authenticates with Edpuzzle using teacher credentials to access assignment media data, including questions and `isCorrect` flags that are not exposed to student accounts. The server handles login and token refresh automatically in the background — you configure the credentials once in `config.json` and the server manages the rest.
  </Accordion>

  <Accordion title="Can I use this on mobile?">
    Bookmarklets are difficult to use on mobile browsers. iOS Safari has limited and cumbersome bookmarklet support. Android Chrome does not support bookmarklets in a user-friendly way. Using a desktop browser will give you a significantly better experience.
  </Accordion>

  <Accordion title="The answers are wrong. What happened?">
    The script reads the `isCorrect` flag directly from Edpuzzle's own API response — it shows exactly what Edpuzzle itself marks as correct. If the displayed answers appear incorrect, verify that you are on the right assignment and that the assignment does not use a randomized question bank, which can shuffle answer choices or vary questions between students.
  </Accordion>
</AccordionGroup>
